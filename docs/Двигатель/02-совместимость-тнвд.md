# 02. Совместимость ТНВД с двигателем 4EB

> Какой ТНВД был штатным для Ford Transit 4EB, какой стоит сейчас, и почему сейчас стоит «не родной».
> Симптомы и пошаговое лечение детонации — в [`docs/Проблемы/01-детонация.md`](../Проблемы/01-детонация.md).

---

## 1. Какой ТНВД был штатным для 4EB

По данным каталогов Bosch и Ford Transit Forum, **родные ТНВД для турбо-Transit поколения Mk5** (`4EB` / `4EC`, 1994–2000) — это:

| Артикул | Тип | Применение |
|---|---|---|
| **`0 460 414 173`** | `VE4/11F2000R686-X` (или R711/R624) | **Основной** ТНВД для Mk5 4EB / 4EC 100 л.с. с LDA |
| **`0 460 414 174`** | `VE4/11F2000R…` | Аналог, поздние варианты |
| `0 460 414 154` | `VE4/11F2000R711` | Mk4 (`4EA`, 1991–1994), на Mk5 встречается реже |
| `0 460 414 155` | `VE4/11F2000R…` | Mk4 (`4EA`), аналог |

Для атмосферных версий (76 л.с. — `4HB` / `4HCX`) шёл другой набор: `0460414145`, `0460414146`, `0460414141` (без LDA — атмосферное исполнение).

> 💡 Если ищется «правильный» б/у ТНВД на 4EB — приоритет **`0460414173` или `0460414174`**. `154` / `155` от Mk4 — могут подойти как «компромиссный вариант», но это уже другая калибровка.

## 2. Какой ТНВД стоит сейчас (донор)

| Артикул | Тип-код | Применение донора |
|---|---|---|
| **`0 460 414 159`** (стоит у меня) | `VE4/11F2000R509-4` | **Land Rover Defender 300Tdi** (`ERR 7299` — военная Wolf-версия; гражданский эквивалент `ERR 4419`) |

Двигатель Land Rover 300Tdi (1994–1998):
- 2,5 л турбодизель прямого впрыска (DI),
- 4 цилиндра, 11 мм плунжер ТНВД,
- 111 л.с. (Discovery / Defender / Range Rover Classic),
- **Турбо С интеркулером**.

### Подтверждение принадлежности 0460414159 к 300Tdi

- **Bob Beck Fuel Injection** (UK, специализированный сервис) — продаёт `0460414159` как «Discovery 300tdi DIESEL FUEL INJECTION PUMP».
- **Mick Ogden Diesels** — таблица Bosch test plans: `VER509` (наш `R509-4`) → `0460414098 / 099` → 300Tdi (ERR4419 / ERR4046).
- **LandyZone форум** (опытный владелец военного Wolf Defender): «Part number for the Wolf injector Pump is **`ERR 7299`** (as stamped on the pump casing) and yet the civilian version is **`ERR4419`**. The 7299 is a later model and may have some uprated internals but the housings are identical and… the fuel settings are also identical.»

Выходит, физически это насос **Land Rover Defender 300Tdi версии Wolf** (военная). Внутренние настройки идентичны гражданскому 300Tdi.

Полные ссылки — в [`docs/Источники/01-ссылки.md`](../Источники/01-ссылки.md), раздел «Land Rover (двигатели-доноры)».

## 4. Почему машина едет — но детонирует

Самый частый вопрос: «Если они разные — почему оно вообще работает?»

Совпадает **6 из 7** параметров совместимости. Это значит, что насос механически совместим, физически крутится в нужную сторону, выдаёт давление в нужный момент, дозирует примерно нужный объём топлива. Двигатель **работает**.

Но седьмой параметр — калибровка — не совпадает. Это даёт **отклонения в режимах**:

```
                ▲
                │
   Цикловая    ┤              ┌─── R509-4 (Land Rover 300Tdi с интеркулером)
   подача      │             /
   мг/такт     │            /
   ────────    │           /  ┌─── R686/R711 (родной для Ford 4EB без интеркулера)
                │          /  /
                │         / /
                │        / /
                │       //
                │      //
                │     //
                │    //
                │   //
                │  //
                │ //
                │//
                └──────────────────────────────►
                              давление наддува
```

То есть Land Rover-калибровка (`R509-4`) добавляет **больше топлива** при том же давлении наддува, чем нужно Ford 4EB.

Дополнительно: **на Land Rover 300Tdi воздух идёт через интеркулер** и охлаждается до ~30–50 °C. На Ford 4EB **без интеркулера** воздух идёт прямо в цилиндры из турбины, **80–100 °C**. Это означает:

- В цилиндр Ford-а приходит **горячий и более разрежённый воздух** (горячий = меньше плотности → меньше реальной массы кислорода).
- ТНВД при этом **подаёт «ладроверовскую» порцию топлива** (как для холодного и плотного воздуха).
- Получается **переобогащённая смесь в горячем воздухе** → горячее пламя + ранний самонагрев заряда → **самовоспламенение начинается раньше плановой ВМТ** → **детонация (стук «пальцев»)**.

Это **не** «насос неисправен». Это **рассогласование калибровки с двигателем**.

## 5. Что усугубляет ситуацию именно на этой машине

| Фактор | Эффект |
|---|---|
| **Нет интеркулера** | Горячий воздух → выше детонационная склонность |
| **Калибровка под 111 л.с., а двигатель 100 л.с.** | На рабочих режимах — больше топлива, чем нужно |
| Степень сжатия 4EB = 18,3 : 1 | На границе допуска для **горячего** воздуха — повышает чувствительность к перебогащению |
| Возможно, изношенные форсунки | Плохой распыл → факел горит «пятном» → локальная детонация |

> 💡 Заметка про степень сжатия: у самого Land Rover 300Tdi сжатие выше (~19,5 : 1), и он работает на этой же калибровке без детонации — потому что у него интеркулер. То есть «высокая степень сжатия» сама по себе **не причина** — причина именно **в комбинации горячего воздуха + калибровка под охлаждённый**.

## 6. Что НЕ проблема (можно успокоиться)

- ✅ Направление вращения — правильное (иначе насос не качал бы топливо вообще).
- ✅ Турбо-исполнение — правильное (LDA-трубка стоит, мембрана работает).
- ✅ Электромагнит отсечки — стандартный 12 В, совместим.
- ✅ Привод и шкив — совместимы (зубчатый ремень одинаковый).
- ✅ Срочное «убитие двигателя за неделю» — **нет**, при щадящей езде. Но если давить газ в пол по горкам и при буксировке — детонация может прогреть донце поршня до прогара. Подробнее — в [`docs/Проблемы/01-детонация.md`](../Проблемы/01-детонация.md).

## 7. Альтернативы — если устранить детонацию настройкой не удастся

| Вариант | Стоимость (ориентир) | Сложность |
|---|---|---|
| **A.** Регулировка max-fuel screw + УОВ (см. [`docs/Проблемы/01-детонация.md`](../Проблемы/01-детонация.md), Шаги 1–2) | 0 € | Низкая |
| **B.** Перекалибровка `0460414159` на стенде под Ford 4EB без интеркулера | 150–250 € | Средняя |
| **C.** Замена форсунок / регулировка давления открытия | 50–150 € | Низкая |
| **D.** Установка штатного `0460414173` / `0460414174` (б/у) | 200–400 € + работа | Средняя |
| **E.** Установка интеркулера (доработка двигателя) | 200–500 € + интеграция | Высокая, но решает много проблем разом |

Предпочтительный порядок: A → B → C → D → E. Подробное обоснование и риски каждого варианта — в [`docs/Проблемы/01-детонация.md`](../Проблемы/01-детонация.md), раздел «Сводная таблица».
