# 01. Характеристики ТНВД 0 460 414 159

> Полная техническая спецификация фактически установленного на машине ТНВД.
> Документ читается отдельно — при чтении не нужны другие файлы проекта.

---

## 1. Идентификационные данные с реальной таблички

На корпусе ТНВД выбита/наклеена табличка с такими маркировками:

| Поле | Значение | Что это значит |
|---|---|---|
| Артикул Bosch | **`0 460 414 159`** | Заводской каталожный номер ТНВД |
| Калибровочный код | **`R509A`** | Тестовый план Bosch — по нему ТНВД настраивается на стенде. Уникален для конкретной модели/мощности двигателя |
| OEM-номер | **`ERR 7299`** | Land Rover OEM — артикул, под которым насос продавался Land Rover-ом |
| Внутренний код | **`770 143242`** | Код даты выпуска / партии Bosch (внутренняя информация Bosch) |
| Серийный № | **`28748`** | Серийный номер конкретного экземпляра |
| Доп. маркировка на корпусе | `bosh 730769162` | **НЕ оригинальный Bosch.** Скорее всего номер ремонтника/восстановителя — насос проходил через дизельный сервис |

> ⚠️ Маркировка `bosh 730769162` (с орфографической ошибкой «bosh» вместо «Bosch») — это почти 100 % признак, что насос **уже был на стенде у дизельщика**. Это важно для понимания, почему он сейчас может быть откалиброван не строго под Land Rover 200Tdi, а под что-то промежуточное.

## 2. Расшифровка типокода (модели)

Тип ТНВД по официальной номенклатуре Bosch — **`VE4/11F2000R...`** (точная конечная цифра без подтверждения второго независимого источника).

Расшифровка по символам:

```
  V E    4    /    11    F     2000      R       (xxx)-(y)
  │ │    │         │     │      │        │         │   │
  │ │    │         │     │      │        │         │   └── ревизия исполнения
  │ │    │         │     │      │        │         └────── конструктивная модификация
  │ │    │         │     │      │        └──────────────── направление вращения
  │ │    │         │     │      │                          R = по часовой (со стороны привода)
  │ │    │         │     │      │                          L = против часовой
  │ │    │         │     │      └───────────────────────── частота вращения вала ТНВД (об/мин),
  │ │    │         │     │                                 по которой настроен регулятор
  │ │    │         │     │                                 ⇒ 2000 об/мин ТНВД = 4000 об/мин
  │ │    │         │     │                                   двигателя (передача 1:2)
  │ │    │         │     └──────────────────────────────── F = Fliehkraftregler =
  │ │    │         │                                       механический центробежный регулятор
  │ │    │         └────────────────────────────────────── диаметр плунжера в мм
  │ │    └──────────────────────────────────────────────── число цилиндров
  │ └─────────────────────────────────────────────────── E = распределительный (Verteiler)
  └───────────────────────────────────────────────────── V = тип насоса (Verteiler)
```

## 3. Главные эксплуатационные характеристики

| Параметр | Значение | Комментарий |
|---|---|---|
| Тип конструкции | Распределительный (VE) | Один плунжер на все 4 цилиндра, аксиально-плунжерная схема с торцевым кулачковым диском |
| Число цилиндров | **4** | Совпадает с двигателем 4EB |
| Диаметр плунжера | **11 мм** | Определяет максимально возможную цикловую подачу |
| Макс. обороты вала ТНВД | ~2000 об/мин | ⇒ макс. обороты двигателя ~4000 об/мин (отсечка) |
| Направление вращения | **По часовой** (со стороны привода) | Совпадает с приводом 4EB |
| Регулятор оборотов | Механический центробежный + **пневматический LDA** | LDA добавляет топливо при росте давления наддува |
| Подкачивающий насос | Пластинчато-роторный, встроенный | Создаёт давление 3–10 бар внутри корпуса ТНВД |
| Опережение впрыска | Гидравлическая муфта | Автоматически смещает момент впрыска по оборотам |
| Электрический клапан отсечки | 12 В постоянного тока | При выключении зажигания перекрывает подачу — двигатель глохнет |
| Управление | Тросовая тяга (без электроники) | Кроме потенциометра обратной связи (поз. 442 на схеме) |
| Тип топлива | Дизельное топливо ДТ | Качественное, без воды и грязи |

## 4. Что есть на этом ТНВД и чего нет

### Есть
- ✅ **LDA-корректор** (поз. 67 на схеме, артикул `1 467 134 399`) — пневматическая мембрана, которой подходит трубка от впускного коллектора. **Это «сердце» турбо-исполнения.**
- ✅ **Регулятор частичных нагрузок** (поз. 68, артикул `1 463 163 150`) — позволяет точнее дозировать топливо в средних режимах
- ✅ **Электромагнит отсечки** (поз. 240, `0 330 001 046`) — 12 В
- ✅ **Потенциометр положения рычага газа** (поз. 442, `1 467 217 359`) — датчик обратной связи на тахометр / диагностический разъём
- ✅ **Полумуфта на валу** (поз. 270, `1 466 434 322`) — для соединения с шестернёй привода

### Нет
- ❌ Электронного блока управления (EDC) — это полностью механический насос
- ❌ ALDA / ALDA-stop (вакуумного выключения наддува-обогащения)

## 5. Принцип работы — простыми словами

ТНВД делает 4 вещи одновременно:

1. **Засасывает солярку из бака** (этим занимается встроенный подкачивающий насос — поз. 7).
2. **Создаёт высокое давление впрыска** (~300–600 бар на выходе из головки) — этим занимается одиночный плунжер, который ездит вверх-вниз и одновременно вращается.
3. **Распределяет это давление по 4 нагнетательным трубкам** к форсункам по очереди — каждый цилиндр получает свою порцию в свой такт.
4. **Дозирует количество топлива** (мг на каждый впрыск) — этим занимается регулятор: учитывает обороты двигателя (центробежные грузики поз. 805+112), положение педали газа (рычаг поз. 72), и **давление наддува** (LDA, поз. 67).

Без LDA это был бы атмосферный ТНВД. С LDA — турбо.

## 6. Что определяет совместимость ТНВД с двигателем

Это не «один ТНВД на все случаи». Совместимость определяется **семью** вещами:

| № | Параметр | Должен совпадать с двигателем |
|---|---|---|
| 1 | Число цилиндров | Точно (4=4) |
| 2 | Направление вращения (R/L) | Точно — насос НЕ работает в обратную сторону |
| 3 | Тип впрыска (DI/IDI) | Желательно совпадать — это влияет на профиль кулачка |
| 4 | Диаметр плунжера | Желательно тот же |
| 5 | Макс. обороты регулятора | Желательно совпадать (или быть выше нужных) |
| 6 | Наличие/отсутствие LDA | На турбо-двигатель — обязательно с LDA |
| 7 | Конкретная калибровка (тест-план Bosch) | Идеально — родной план; на практике — близкий план |

В нашем случае пункты 1–6 совпадают, **пункт 7 — нет** (R509A — это план Land Rover 200Tdi с интеркулером, а не Ford Transit 4EB без интеркулера). Это и даёт детонацию — подробнее в [`02-vehicle-and-compatibility.md`](./02-vehicle-and-compatibility.md).
