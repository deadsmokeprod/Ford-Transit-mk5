## 1. Главные эксплуатационные параметры

| Параметр | Значение | Краткое пояснение |
|---|---|---|
| Артикул Bosch | **`0 460 414 159`** | Заводской каталожный номер |
| Полный тип-код | **`VE4/11F2000R509-4`** | Расшифровка — в [`02-маркировка.md`](./02-маркировка.md) |
| Тип конструкции | Распределительный (VE), аксиально-плунжерный | Один плунжер обслуживает все 4 цилиндра |
| Число цилиндров | **4** | Точно совпадает с двигателем 4EB |
| Диаметр плунжера | **11 мм** | Определяет максимально возможную цикловую подачу топлива |
| Макс. обороты вала ТНВД | ~2000 об/мин | Передача 1:2 ⇒ макс. обороты двигателя ~4000 об/мин (отсечка) |
| Направление вращения | **R = по часовой** (со стороны привода) | В обратную сторону насос работать не будет |
| Регулятор оборотов | Механический центробежный + **пневматический LDA** | LDA добавляет топливо при росте давления наддува |
| Подкачивающий насос | Пластинчато-роторный, встроенный | Создаёт давление 3–10 бар внутри корпуса ТНВД |
| Опережение впрыска | Гидравлическая муфта внутри ТНВД | Автоматически смещает момент впрыска по оборотам |
| Электрический клапан отсечки | 12 В постоянного тока | При выключении зажигания перекрывает подачу — двигатель глохнет |
| Управление | Тросовая тяга (без электроники) | Кроме потенциометра обратной связи — поз. 442 на схеме |
| Тип топлива | Дизельное топливо (ДТ) | Качественное, без воды и грязи |

## 2. Что есть на этом ТНВД и чего нет

### Есть

| Узел | Где на схеме | Зачем нужен |
|---|---|---|
| **LDA-корректор** (пневматическая мембрана) | поз. 67 (артикул `1 467 134 399`) | Подключается трубкой к впускному коллектору. Обогащает смесь при наддуве. Это «сердце» турбо-исполнения насоса |
| Регулятор частичных нагрузок | поз. 68 (артикул `1 463 163 150`) | Позволяет точнее дозировать топливо в средних режимах |
| Электромагнит отсечки | поз. 240 (артикул `0 330 001 046`) | 12 В, глушит двигатель при выключении ключа |
| Потенциометр положения рычага газа | поз. 442 (артикул `1 467 217 359`) | Датчик обратной связи на тахометр / диагностический разъём |
| Полумуфта на валу | поз. 270 (артикул `1 466 434 322`) | Соединение с шестернёй привода |

### Нет

- ❌ Электронного блока управления (EDC) — насос полностью механический.
- ❌ ALDA / ALDA-stop (вакуумного выключения наддув-обогащения).

## 3. Принцип работы простыми словами

ТНВД делает 4 вещи **одновременно**:

1. **Засасывает солярку из бака** — этим занимается встроенный подкачивающий насос (поз. 7 на схеме).
2. **Создаёт высокое давление впрыска** (~300–600 бар на выходе из головки) — этим занимается одиночный плунжер, который ездит вверх-вниз и одновременно вращается.
3. **Распределяет это давление по 4 нагнетательным трубкам** к форсункам по очереди — каждый цилиндр получает свою порцию в свой такт.
4. **Дозирует количество топлива** (мг на каждый впрыск) — этим занимается регулятор: учитывает обороты двигателя (центробежные грузики поз. 805+112), положение педали газа (рычаг поз. 72) и **давление наддува** (LDA, поз. 67).

Без LDA это был бы атмосферный ТНВД. С LDA — турбо-исполнение.

## 4. Что определяет совместимость ТНВД с двигателем

Это не «один ТНВД на все случаи». Совместимость определяется **семью** параметрами:

| № | Параметр | Должен совпадать с двигателем |
|---|---|---|
| 1 | Число цилиндров | Точно (4 = 4) |
| 2 | Направление вращения (R/L) | Точно — насос НЕ работает в обратную сторону |
| 3 | Тип впрыска (DI/IDI) | Желательно совпадать — это влияет на профиль кулачка |
| 4 | Диаметр плунжера | Желательно тот же |
| 5 | Макс. обороты регулятора | Желательно совпадать (или быть выше нужных) |
| 6 | Наличие/отсутствие LDA | На турбо-двигатель — обязательно с LDA |
| 7 | Конкретная калибровка (тест-план Bosch) | Идеально — родной план; на практике — близкий план |

В нашем случае пункты 1–6 совпадают, **пункт 7 — нет**: тест-план `R509-4` рассчитан на Land Rover 300Tdi с интеркулером, а двигатель Ford 4EB интеркулера не имеет. Это и порождает детонацию — подробности и пошаговое лечение в [`docs/Проблемы/01-детонация.md`](../Проблемы/01-детонация.md).
