# 02. Автомобиль, двигатель и совместимость

> Подробно: что за двигатель 4EB у владельца, какой ТНВД был «штатным» с завода, почему сейчас стоит «не родной», и почему это даёт детонацию.

---

## 1. Двигатель Ford Transit 4EB (1995 model year)

### Точные характеристики (из официального Ford Workshop Manual)

| Параметр | Значение |
|---|---|
| Семейство | Ford Transit Mk5 (08/1994 – 07/2000) |
| Тип двигателя | **2.5 TD — турбодизель прямого впрыска (DI)** |
| Заводской код | **`4EB`** для модельного года 1995 |
| Объём | 2496 см³ |
| Цилиндров | 4 рядно |
| Диаметр × ход поршня | 90,54 × 93,67 мм |
| Степень сжатия | **18,3 : 1** |
| Мощность (EEC) | **74 кВт = 100 л.с.** при 4000 об/мин |
| Крутящий момент (EEC) | **224 Н·м** при 2100 об/мин |
| Холостые обороты | 850 ± 50 об/мин |
| Макс. обороты постоянно | 4000 об/мин |
| Макс. обороты кратковременно | 4400 об/мин |
| Порядок впрыска | 1-2-4-3 |
| Статический УОВ (опережение впрыска) | **11° до ВМТ** |
| **Турбонаддув** | **Есть** (Garrett GT, обычно ~0.7–0.9 бар на полной нагрузке) |
| **Интеркулер** | **НЕТ в этой комплектации** ⚠️ |

> ⚠️ **Отсутствие интеркулера — ключевой фактор**, к которому мы будем возвращаться. Без интеркулера наддувочный воздух поступает в цилиндры **горячим (~80–100 °C)**. Чем горячее воздух — тем сильнее склонность к детонации. Это **усиливает** проблему «жирной» калибровки ТНВД.

## 2. Какой ТНВД был **штатным** для 4EB

По данным каталогов Bosch и Ford Transit Forum, **родные ТНВД для турбо-Transit (4EB / 4EC)** — это:

| Артикул | Тип | Применение |
|---|---|---|
| **`0 460 414 154`** | `VE4/11F2000R711` | Для турбо 100 л.с. с LDA |
| `0 460 414 155` | `VE4/11F2000R...` | Аналог |
| `0 460 414 173` | `VE4/11F2000R...` | Поздние варианты |
| `0 460 414 174` | `VE4/11F2000R...` | Поздние варианты |

Для атмосферных версий (76 л.с. — `4HB`) шёл другой набор: `0460414145`, `0460414146`, `0460414141`.

---

## 3. Какой ТНВД стоит **сейчас** (донор)

| Артикул | Тип | Применение донора |
|---|---|---|
| **`0 460 414 159`** (стоит у вас) | `VE4/11F2000R...` (близкий) | **Land Rover 200Tdi / 300Tdi** (Defender 90/110, Discovery, Range Rover Classic, 1989–1998) |

Двигатель Land Rover 200Tdi:
- 2.5 TD прямой впрыск (DI)
- 4 цилиндра, 11 мм плунжер ТНВД
- ~111 л.с. при 4000 об/мин
- **Турбо С интеркулером**

Соответствие подтверждается каталогом Goldfarb & Associates: `0-460-414-159DR (ERR1299; ERR7299) Rebuilt Bosch VE4 Injection Pump fits Landrover Engine`.

## 5. **Почему машина едет** — но детонирует

Это самый частый вопрос: «Если они разные — почему оно вообще работает?»

Ответ простой: совпадает **6 из 7** параметров совместимости. Это значит, что насос механически совместим, физически крутится в нужную сторону, выдаёт давление в нужный момент, дозирует примерно нужный объём топлива. Двигатель **работает**.

Но седьмой параметр — калибровка — не совпадает. Это даёт **отклонения в режимах**:

```
                ▲
                │
   Цикловая    ┤              ┌─── R509A (Land Rover 200Tdi с интеркулером)
   подача      │             /
   мг/такт     │            /
   ────────    │           /  ┌─── R711 (родной для Ford 4EB без интеркулера)
   ────────    │          /  /
                │         / /
                │        / /
                │       //
                │      //
                │     //
                │    //
                │   //
                │  //
                │ //
                │//
                └──────────────────────────────►
                              давление наддува
```

То есть Land Rover-калибровка добавляет **больше топлива** при том же давлении наддува, чем нужно Ford 4EB.

Дополнительно: **на Land Rover воздух идёт через интеркулер** и охлаждается до ~30–50 °C. На Ford 4EB **без интеркулера** воздух идёт прямо в цилиндры из турбины, **80–100 °C**. Это означает:

- В цилиндр Ford-а приходит **горячий и более разрежённый воздух** (горячий = меньше плотности → меньше реальной массы кислорода)
- ТНВД при этом **подаёт «ладроверовскую» порцию топлива** (как для холодного и плотного воздуха)
- Получается **переобогащённая смесь в горячем воздухе** → высокое давление сжатия + горячее → **самовоспламенение начинается раньше плановой ВМТ** → **детонация (стук «пальцев»)**

Это не «насос неисправен». Это **рассогласование калибровки с двигателем**.

---

## 6. Что усугубляет ситуацию именно у нас

| Фактор | Эффект |
|---|---|
| **Нет интеркулера** | Горячий воздух → выше детонационная склонность |
| **Калибровка под 111 л.с., а двигатель 100 л.с.** | Подача на ~10 % больше нужной |
| **Степень сжатия 4EB = 18,3:1** (высокая) | Ещё больше склонность к детонации |
| **Возможно, изношенные форсунки** | Плохой распыл → факел горит «пятном» → локальная детонация |

---

## 7. Что **не** проблема (и можно успокоиться)

- ✅ Направление вращения — правильное (иначе насос не качал бы топливо вообще)
- ✅ Турбо-исполнение — правильное (LDA-трубка стоит, мембрана работает)
- ✅ Электромагнит отсечки — стандартный 12 В, совместим
- ✅ Привод и шкив — совместимы (зубчатый ремень одинаковый)
- ✅ Полное и срочное «убитие двигателя за неделю» — **нет**, при щадящей езде

Подробно как лечить детонацию (без замены насоса) — в [`04-troubleshooting.md`](./04-troubleshooting.md).

## 8. Альтернативы: если устранить детонацию настройкой не удастся

| Вариант | Стоимость (ориентир) | Сложность |
|---|---|---|
| **A. Регулировка max-fuel screw + УОВ** | 0 € | Низкая |
| **B. Перекалибровка `0460414159` на стенде под Ford 4EB без интеркулера** | 150–250 € | Средняя |
| **C. Замена форсунок / регулировка давления открытия** | 50–150 € | Низкая |
| **D. Установка штатного `0460414154` (б/у)** | 200–400 € + работа | Средняя |
| **E. Установка интеркулера (доработка)** | 200–500 € + интеграция | Высокая, но решит много проблем разом |
